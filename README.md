Данный инструмент предназначен для анализа и визуализации графа
зависимостей пакетов. Код написан на языке Pyhton. Для запуска
тестирования созданы тестовые скрипты для каждого
этапа.


Ход разработки по этапам:
Этап 1:
    Добавлена поддержка INI конфигурационных файлов с обязательными параметрами:
        * Имя пакета
        * URL репозитория или путь к тестовому файлу
        * Флаг тестового режима
        * Версия пакета
        * Имя выходного файла
    - Вывод всех настраиваемых параметров в формате ключ-значение
    - Реализована комплексная обработка ошибок для всех параметров
    - Создана базовая структура CLI приложения
    - Добавлены тестовые скрипты для нескольких конфигурационных файлов
    - Создана документация README.md

Этап 2:
    - Добавлена интеграция с npm registry для получения данных о зависимостях
    - Реализовано извлечение прямых зависимостей для указанной версии пакета
    - Вывод всех прямых зависимостей в соответствии с требованиями этапа
    - Обработка ошибок HTTP запросов и проблем с подключением
    - Упрощена структура приложения, удален тестовый режим
    - Используется только реальный npm registry для получения данных

Этап 3:
  - Реализован алгоритм DFS без рекурсии для построения графа зависимостей
  - Добавлена обработка циклических зависимостей
  - Реализована поддержка тестового режима с файлами графов
  - Упрощен вывод параметров конфигурации
  - Убраны версии пакетов из вывода зависимостей
  - Автоматический выбор пакета из графа при необходимости
  - Улучшена обработка ошибок и валидация графов

Этап 4:
  - Добавлена функция поиска обратных зависимостей с использованием DFS
  - Обратные зависимости ищутся только для целевого пакета
  - Реализовано построение графа обратных зависимостей
  - Обновлены тестовые сценарии для демонстрации функциональности

Этап 5:
  - Генерация Mermaid кода для визуализации графа зависимостей
  - Вывод чистого Mermaid кода в консоль
  - Поддержка стилей для корневого пакета
  - Формирование связей между пакетами в формате Mermaid
  - Интеграция визуализации в основной процесс работы программы
  - Сохранение всей предыдущей функциональности
  - Корректная работа с онлайн редакторами Mermaid

    Примеры визуализации зависимостей для трех различных пакетов, созданные
с помощью сайта https://mermaid.live/edit, представления этих
графов на языке диаграмм Mermaid создано в файле "run_test5.bat"
